# Hierarchy-Icon

This package can show the game object component icons in the hierarchy and customize the icons shown parameter.

## Installation

Window -> Package Manager -> + -> Add package from git URL

paste git url:

https://github.com/Ryuu-64/Hierarchy-Icon.git#main

or

https://e.coding.net/ryuu64/hierarchy-icon/Hierarchy-Icon.git#main

click add button


## Customize

Check the ScriptableObject HierarchyIconInfo at path:

Assets\Plugins\Ryuu\Hierarchy Icon\HierarchyIconInfo.asset
